# Welcome to VeilforgeSMP

**VeilforgeSMP** is a Minecraft survival experience built for players who enjoy **PvP, economy, raiding, grinding, and long-term progression**.  
The server is inspired by the fast-paced style of **DonutSMP**, while adding its own custom feel and progression system.

## Server Info

- **Server Name:** VeilforgeSMP
- **IP Address:** `play.veilforge.fun`
- **Game Version:** `1.20.x – 1.21.11`
- **Mode:** Survival / PvP / Economy

## World Borders

To keep the world fresh and encourage exploration, borders are managed and expanded over time.

- **Overworld:** 20,000 blocks
- **Nether:** 7,000 blocks
- **End:** 4,000 blocks

Borders may expand every **2–3 months** depending on server progression and community activity.

## World Progression

To make the server lifecycle more exciting, certain dimensions unlock after launch:

- **Nether opens:** 3 days after server launch
- **End opens:** 1 week after server launch

This helps keep early-game progression balanced and gives all players a fair start.

## Features

- Inspired by classic hardcore SMP grinding
- Balanced early-game progression
